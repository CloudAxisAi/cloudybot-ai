# Example: Competitor Research with CloudyBot

Full competitive analysis in one task.

**Manual time:** Half a day | **With CloudyBot:** ~20 minutes

---

## The Task

```
Research the top 5 competitors of [your product/category].

For each competitor, find:
1. Product name and website
2. Pricing (all tiers if available)
3. Key features (top 5-7)
4. Target audience
5. One-sentence positioning statement
6. Notable customer complaints from G2 or Capterra

Format as a comparison table, then add a "Gaps & Opportunities" section.
```

---

## What CloudyBot Does

1. Searches Google for top competitors
2. Visits each website and pricing page
3. Checks G2/Capterra for review summaries
4. Compiles into a structured report

---

## Chain It

```
Based on the competitor analysis above, write a positioning statement for CloudyBot
that highlights our 3 biggest differentiators.
```

```
Write 5 landing page headline variations that directly address competitor weaknesses.
```

Context carries forward — no re-explaining needed.
