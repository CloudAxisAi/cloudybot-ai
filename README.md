## What is CloudyBot?

CloudyBot is a **hosted AI agent** that runs in the cloud with a real browser. You give it tasks via chat or WhatsApp — it browses, clicks, reads, scrapes, fills forms, and reports back.

No installs. No API keys. No config files. No surprise bills.

---

## Key Features

| Feature | Description |
|---|---|
| **Cloud Browser** | A real browser runs in our cloud. CloudyBot navigates sites, fills forms, scrapes data — just like a human assistant would. |
| **WhatsApp & Chat** | Send tasks from WhatsApp or the web chat. CloudyBot works while you're away. |
| **LinkedIn Automation** | Research profiles, send connection requests, draft outreach messages — hands-free. |
| **Knowledge Base** | Upload PDFs, docs, or paste content. Ask questions and get accurate answers from your own data. |
| **Web Research** | "Find me the top 10 SaaS tools for X" — CloudyBot browses and summarises. |
| **Hard-capped Pricing** | Set a monthly cap. It never exceeds it. No bill shock. |

## How It Works

```
You → Send task via Chat or WhatsApp
         ↓
    CloudyBot AI interprets the task
         ↓
    Cloud Browser executes (browse / click / scrape / fill)
         ↓
    Result delivered back to you
```

No local install. No Puppeteer setup. No Playwright config. It just works.
